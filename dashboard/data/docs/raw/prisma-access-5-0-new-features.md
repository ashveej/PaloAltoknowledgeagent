# New Features in Prisma Access 5.0 and 5.0.1

## App Acceleration in Prisma Access
Released November 2023. Enhances throughput for users experiencing poor app performance due to wireless issues or network congestion. The system learns device type, network conditions, and application context to optimize data flow in real-time, delivering marked throughput improvement for TCP traffic when connecting through Prisma Access compared to direct internet access. Performance gains are measurable via Autonomous DEM metrics.

## App Acceleration Support for Additional Apps
May 2024 release extending acceleration to dynamic content in major SaaS platforms including AWS S3, Azure Storage, Box, Google Drive, OneDrive, Salesforce, SAP Ariba, ServiceNow, Slack file downloads, and Zoom downloads. Offers up to five times the improvement over direct-to-internet app performance in response time and throughput.

## BGP MRAI Configuration Support
Introduced November 2023. Provides configurable Minimum Route Advertisement Interval settings, allowing administrators to balance network convergence speed against update overhead.

## Consistent User Identity Formats
February 2024. Applies global normalization standards to usernames from various identity sources, converting inconsistent formats (mixed case, domain prefixes/suffixes) into unified standardized formats. Example: test.User@abc.com, abc/Test.User, and abc//Test.useR all normalize to test.user.

## IP Optimization for Mobile Users
Released June 2024 for GlobalProtect deployments. Reduces the quantity of public IP addresses requiring management, simplifies allow-listing, improves resilience, and expedites onboarding of new tenants.

## Saudi Arabia Compute Location
June 2024 introduces dedicated infrastructure improving data locality and latency for regional users in the Kingdom of Saudi Arabia.

## Additional Features
Service Connection Identity Redistribution Management (Nov 2023), Strata Cloud Manager support for Prisma Access China (Nov 2023), Strata Logging Service Switzerland region support (Nov 2023), and Traffic Replication for Remote Networks and Strata Cloud Manager (Nov 2023).
